# Strong Woman Camp Poster

This is a web-based poster for Strong Woman Camp that can be hosted on GitHub Pages.

## Setup Instructions

1. Create a new repository on GitHub (e.g., `strong-mama-camp-poster`)
2. Push this code to GitHub:
   ```bash
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/strong-mama-camp-poster.git
   git push -u origin main
   ```
3. Go to your repository on GitHub → Settings → Pages
4. Under "Source", select "Deploy from a branch"
5. Choose "main" branch and "/ (root)" folder
6. Click Save
7. Your poster will be live at: `https://YOUR_USERNAME.github.io/strong-mama-camp-poster/`

That's it! Anyone can now access your poster via the link.
